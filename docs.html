📊 CSV Analyzer Pipeline

Drop any CSV for instant statistical analysis.

# Basic usage
python3 analyze.py sales.csv

# With options
python3 analyze.py data.csv --output report.csv --charts

# Batch mode
python3 analyze.py *.csv --output-dir ./reports

Output: Stats per column (min, max, mean, sum, missing), auto-generated charts, dashboard-ready CSV.
